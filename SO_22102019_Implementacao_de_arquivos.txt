		-> Um sistema de arquivos com journaling é aquele que mantém um log (journal), normalmente circular, de todas as mudanças no sistema de arquivos antes de escrever os dados no disco. Este tipo de sistema de arquivos oferece uma melhor probabilidade de não sofrer corrupção de dados no caso de o sistema travar ou faltar energia, e uma recuperação mais rápida, porque não há necessidade de verificar todo o disco, somente aqueles que pertenciam a um log que não foi devidamente fechado.
